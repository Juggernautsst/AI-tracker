#AI-Tracker


本项目旨在利用区块链技术实现对人工智能项目的监管与审计。


##完整工作流##

1. 初始化与密钥生成


2. AI模型数据收集与摘要生成


3. 对AI摘要进行签名


4. 加密AI摘要


5. 将加密的AI摘要上传到IPFS


6. 将IPFS哈希和签名上传到区块链


7. 验证AI摘要
